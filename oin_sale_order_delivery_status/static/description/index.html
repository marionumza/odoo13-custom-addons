Delivery Status on Sales Order

Delivery Status:

  •   Nothing to Deliver : if no delivery order is created
  •   To Deliver: Delivery order is still to Deliver
  •   Partially Deliver: If Multiple Delivery Order for one sales order and some order are delivered and some are still to deliver.
  •   Delivered: All Order are done
  •   Processing: If Delivery order is in waiting for another operation state (purchase or manufacturing)
